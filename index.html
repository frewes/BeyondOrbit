Beyond Orbit

In celebration of the 2018-2019 Into Orbit theme, we've designed a whole new set of challenges for teams to try out! Build instructions were generated using the LEGO Digital Designer. All parts (except for some in M6 Treadmill) are available in one Into Orbit kit. All models were designed from scratch. Instructions are available online (link in each image) or as downloadable PDFs.

M1 Escape Velocity

Launching into orbit is a precise operation, if you go too slow you crash but too fast and you don’t reach an orbit.

M2 Crater Crossing

Space robots need to be able to cross difficult terrain.

M3 Controlled Descent

When landing on a planet surface, spaceships need to control their descent so they don’t land too hard and break everything.

M4 Tool Use

Robots in space don’t have human hands, so they have to be prepared to autonomously operate equipment offworld.

M5 Spacewalk Rescue

Staying tethered during spacewalks is important; if anything goes wrong, you can be rescued.

M6 Treadmill

Astronauts have to stay fit; astronaut Sunita Williams ran the Boston Marathon on the ISS while her sister ran it in Boston.

M7 Long Range Communication

Space communications are an important issue; it takes time for the signal to travel.

M8 Airlock

If a controlled environment has to be maintained, you have to be very careful when a vacuum is involved.
